TROUBLESHOOTING

Top 5 Erros e Soluções

error

ModuleNotFoundError

Verificar PYTHONPATH e imports relativos. Usar pip install -e . ou ajustar sys.path.

error

StepNotImplemented

Verificar match exato entre frase do .feature e decorator do step. Espaços e acentos contam.

error

MagicMock assert_called falhando

Verificar se mock foi injetado no objeto correto. Usar patch no módulo que importa, não no que define.

error

Circuit Breaker nunca abre

Verificar failure_threshold e reset_timeout. O número de falhas consecutivas precisa atingir o limiar.

error

Locust ConnectionError

Verificar se Flask/servidor está rodando na porta correta. Testar com curl localhost:5000 antes.

QUALIDADE

Padrão de Qualidade Esperado

cancel Evite 

Feature: Testar pagamento
  Scenario: Teste
    Given sistema funcionando
    When faço pagamento
    Then funciona
# Step genérico
@when('faço pagamento')
def step_pagar(context):
    result = MagicMock()  # Mock vazio
    assert result  # Assert inútil

check_circle Faça 

@rf @pagamento @sucesso
Feature: Processar pagamento PIX
  Scenario: PIX confirmado em < 5s
    Given um pedido de R$ 150.00
    And o gateway PIX está disponível
    When solicito pagamento via PIX
    Then pagamento confirmado em < 5 segundos
    And comprovante gerado com transação única
@when('solicito pagamento via PIX')
def step_pix(context):
    context.gateway.processar_pix.return_value = {
        'status': 'aprovado',
        'transacao_id': 'TXN-001',
        'tempo_ms': 1200
    }
    context.resultado = context.gateway.processar_pix(
        valor=context.valor
    )
